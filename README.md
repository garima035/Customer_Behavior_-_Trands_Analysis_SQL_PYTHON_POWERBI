# Customer_Behavior_-_Trands_Analysis_SQL_PYTHON_POWERBI
This Project analyzes customer shopping data to identify trends, understand customer behavior, and improve business decisions. It includes data cleaning, analysis, and visualization using tools like python, SQL, and Power BI


## Project Overview

This project analyzes customer shopping behavior using transactional data from 3,900 purchases across different product categories. The aim is to understand customer preferences, spending patterns, product demand, subscription behavior, and the impact of discounts on sales.

## Objectives

* Identify customer spending trends
* Compare subscriber vs non-subscriber behavior
* Analyze discount and promo code impact
* Find top-rated and most purchased products
* Segment customers into New, Returning, and Loyal groups
* Create visual dashboards for business insights

## Tools Used

* Python (Pandas)
* PostgreSQL / SQL
* Power BI
* GitHub

## Project Files

* Python scripts for data cleaning and analysis
* SQL queries for business insights
* Power BI dashboard files
* Project report and presentation

## Key Insights

* Subscribers spend more than non-subscribers
* Loyal customers generate higher revenue
* Discounts increase sales for selected products
* Certain age groups contribute more to total revenue
* Top-rated products can be used for better marketing campaigns

## Business Recommendations

* Improve loyalty programs
* Promote subscription plans
* Highlight top-selling products
* Use targeted marketing for high-value customers
* Balance discount strategies with profit margins
